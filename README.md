# terapias
Quantas vezes você adiou o seu próprio cuidado? Cada sessão aqui é um convite para parar, de verdade, e sentir o corpo soltar o que já não precisa mais carregar.  Os aromas chegam antes do toque. O tempo muda de ritmo. E você sai de uma forma que é difícil de explicar, mas fácil de sentir.
